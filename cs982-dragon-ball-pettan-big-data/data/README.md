# Data

This project uses a small tabular dataset of Dragon Ball Z Pettan Battle card metadata and statistics. The repository includes a cleaned CSV used for reproducible analysis:

```text
data/processed/pettan_cards_clean.csv
```

The dataset contains card-level fields such as title, character, series, rarity, type, HP, attack, release dates and derived modelling features.

No card images, private files, spreadsheets, PDFs or screenshots are included. The data is used only for educational analysis and is not affiliated with Bandai, Toei Animation or the Dragon Ball franchise.
